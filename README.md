# proyecto_python
Este proyecto consiste en un programa en Python que calcula el puntaje total de un jugador a partir de diferentes acciones realizadas durante un juego, como puntos obtenidos, bonificaciones y penalizaciones. El sistema suma y resta los valores correspondientes para mostrar el puntaje final. Es ideal para practicar operaciones matemáticas, condicionales, funciones y entrada de datos por teclado.
## técnologias utilizadas
- Python

Se utilizan principalmente las siguientes operaciones y comandos:

-  Operaciones

Suma (+) → para agregar puntos y bonificaciones.

Resta (-) → para aplicar penalizaciones.

Asignación (=) → para guardar valores en variables.

Operaciones aritméticas básicas → cálculo del puntaje total.

-  Comandos y estructuras

input() → para ingresar datos del jugador (puntos, penalizaciones, etc.).

int() o float() → para convertir los datos ingresados a números.

Variables → para almacenar los puntajes.

print() → para mostrar el puntaje final en pantalla.

if / else (opcional) → para validar condiciones o aplicar reglas especiales.

Funciones (def) (opcional) → para organizar el cálculo del puntaje.
